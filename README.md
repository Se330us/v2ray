# Ох, ёлки-палки
## Android
Для работы скачиваем приложение v2rayTun → [ссылка на скачивание](https://github.com/DigneZzZ/v2raytun/releases/download/5.19.64/v2RayTun_universal.apk).
Устанавливаем. Есть вдруг ранее не устанавливались сторонние приложения, то телефон ругнется, что надо в настройках включить "установку из небезопасных источников". Делаем это, устанавливаем и идем дальше.
Нас встречает большая серая кнопка по середине.  
картинка  
Сверху справа три точки. Нажимаем их, выбираем импортировать из буфера обмена.  
картинка  
Вставится список из серверов. Зажимаем по его заголовку (по умолчанию это github.com) и нажимаем "Проверить время отклика".  
картинка  
Как только всё проверится напротив каждого сервера будет значение в милисекундах - это задержка соединения. Чем она меньше, тем лучше. Если напротив теста стоит "?" - сервер бракованый.  
картинка  
Нажимаем вновь три точки сверху и нажимаем "Удалить нерабочие конфиги"  
картинка  
Всё. Теперь выбираем понравившийся сервер и пользуемся.
